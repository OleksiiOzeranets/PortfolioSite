ColorScheme web app

JavaScript & API

GitHub Repo Live version

This app utilizes HTML, CSS, and JavaScript to allow users to generate color schemes based on a selected color and scheme type. The app fetches data from The Color API using the Fetch API, dynamically displays the colors, and enables users to copy a color's hex code to the clipboard. The app also incorporates event listeners for interactivity and DOM manipulation to update the content displayed on the page.

Blog Project web app

NodeJS & MongoDB

GitHub Repo Live version

This is a full-stack Blog application built using Node.js, Express, MongoDB, and EJS. It allows users to create, read, and delete blog posts through a clean, server-rendered interface. MongoDB serves as the database to store blog data, while Express handles routing and business logic. EJS is used to dynamically render HTML templates with data.
